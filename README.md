# 🚚 동네비서 (dnbsir.com) - 똑똑한 AI 친구

강원도 태백 현장의 소상공인과 농민을 돕기 위한 AI 에이전트 서비스입니다.

## 🚀 시스템 현황
* **엔진**: Google Cloud Run (Serverless)
* **서버 위치**: 미국 (`us-central1`)
* **현장 기지**: 강원도 태백시
* **주요 기능**: 부재중 전화 응대, 택배 접수 자동화, 단골 관리

## 🛠️ 기술 스택
* **Language**: Python 3.9
* **Framework**: FastAPI (로컬 포트 8080)
* **Database**: Google Sheets API 연동
* **Deployment**: GitHub Auto-deploy to Google Cloud
