### 기존 기술 대비 차별점
1. 기존 기술은 가맹점주 스마트폰 앱 실행 상태에 의존  
   → 본 기술은 통신사/가상번호 망 신호 기반 서버 처리로 단말 무관
2. 기존 기술은 문자 발송만 단순 수행  
   → 본 기술은 주문·결제·물류(택배/배달)까지 연동된 O2O 파이프라인 구현
3. 기존 기술은 점주 개인 요금/문자 건수에 종속  
   → 본 기술은 서버 충전금 기반 정산 엔진으로 비용 분리

### 핵심 독창성(특허 포인트)
부재중/통화중 신호를 네트워크 단에서 감지하여, 가맹점주의 단말 상태와 무관하게 중앙 서버가 주문 링크를 자동 발송하고, 주문/결제/물류로 이어지는 후속 파이프라인을 자동 실행하는 시스템.

2. **서버 개입:** 050 가상번호 서버가 해당 신호를 실시간 인식
3. **매칭 로직:** 발신번호 ↔ 가맹점 고유 주문 URL 매핑 조회
4. **메시지 발송:** 서버 메시징 엔진으로 고객에게 알림톡/문자 자동 발송
